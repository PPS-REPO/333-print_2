# 문제

프로그래밍에는 입출력이라는게 있다.
그중에서 우리는 처음으로 출력을 해볼것이다.
이 문제는 두번째로 배우는 프로그래밍 문제로서 "Hello!"를 두개의 줄에 걸쳐 출력하는 문제다.

# 입력

입력은 없다.

# 출력

"Hello!"를 두번에 걸쳐 출력해보자.

python 에선
```py
print("Hello!Hello!")
```

C++ Or C에선

```c
#include <stdio.h>
int main(){
  printf("Hello!Hello!");
  return 0;
}
```
으로 이 문제를 통과 할 수 있다.
